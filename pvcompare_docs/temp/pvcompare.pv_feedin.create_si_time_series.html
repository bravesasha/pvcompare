pvcompare.pv_feedin.create_si_time_series¶

pvcompare.pv_feedin.create_si_time_series(lat, lon, weather, surface_azimuth, surface_tilt, normalized=False)[source]¶

Calculates feed-in time series for a silicon PV module.

The cpv time series is created for a given weather data frame, at a given orientation for the flat plate module ‘Canadian_Solar_CS5P_220M___2009_’. The time series is normalized by the peak power of the module.

Parameters

  • lat (float) – latitude

  • lon (float) – longitude

  • weather (pandas.DataFrame) –

  • surface_azimuth (float) – surface azimuth of the modules

  • surface_tilt (float) – surface tilt of the modules

  • normalized (boolean) –

Returns

Return type

pandas.Series
