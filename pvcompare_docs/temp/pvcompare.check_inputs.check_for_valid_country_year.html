pvcompare.check_inputs.check_for_valid_country_year¶

pvcompare.check_inputs.check_for_valid_country_year(country, year, input_directory)[source]¶

The inputs checkes the countries for population, workalender and consumption.

Parameters

  • country (str) – country of interest

  • year (int) – year of interest

  • input_directory (str) – input directory for data input
